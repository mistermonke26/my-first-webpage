# Joke Reveal Webpage

A simple single-page website built with plain HTML, CSS, and JavaScript.

## What this project includes

- Proper `index.html` structure (`<!DOCTYPE html>`, `head`, `body`)
- A heading and fun content with emojis
- Custom CSS styling (layout, colors, spacing, and button styles)
- Interactive feature: click a button to reveal/hide the joke punchline

## Joke interaction

- Joke setup is shown by default
- Clicking **Reveal punchline** shows the answer
- Clicking **Hide punchline** hides it again

## Local development

1. Open the project folder.
2. Open `index.html` in your browser.

## Deployment

This repo is intended to deploy automatically with Vercel when changes are pushed to GitHub.
